Add an event

This tool helps you add events to the website.

About this tool

Complete this form to generate the text you need to upload content to the website.

Enter the information into the form below, then select the output you require.

The fastest way to add posts to the website is to use GitHub. If you have a GitHub account, you can add files directly to our GitHub repository via a pull request. The tool can step you through the process of creating a new post directly.

If you have difficulties using GitHub, you can also send the information via email. The tool can:

  1. generate an email to send to our central team to post the information
  2. generate a file that you can send to our central team via email to post
  3. copy the information the central team will need to your clipboard; you can then paste this information into an email to send to our central team to post

This form might not work on your departmental IT system.

There is nothing we can do to fix this. If you experience problems, try using a personal device instead.

Step 1 of 4

Open GitHub in a new window

Use the button below to open GitHub in a new window, at the correct directory.

If you are asked to "fork this repository", read our guidance on using GitHub.

Step 2 of 4

Copy and paste the filename into GitHub

Click the button below to copy the file name to your clipboard.

Paste the file name into the "Name your file..." field in the GitHub window.

Step 3 of 4

Copy and paste the post content into GitHub

Click the button below to copy the post content to your clipboard.

Paste the post content into the large text field in the GitHub window.

Step 4 of 4

Commit the file

On the GitHub window, select "Commit new file", at the bottom of the page.

Alternative methods

Use these options only if you can't upload via GitHub.

Generate an email

Click the button below to generate an email containing the post content.

Send the email unedited to info@civilservice.lgbt.

Download as a file

Click the button below to download the post content as a plain text file in Markdown format.

Attach the file to an email and send it to info@civilservice.lgbt.

If you need to edit the content of the post after you have downloaded it, you can open the file using TextEdit on a Mac or Notepad on a Windows PC.

Copy to clipboard

Click the button below to copy the post content to your clipboard.

Paste the information into an email and send it to info@civilservice.lgbt.
